Krapperup's farm museum

These web pages provide translations for the Swedish text shown in the various posters in the museum. At each poster a QR code can be used to access the poster's content in alternative languages. Each poster is numbered and these numbers can be used as an alternative way to find the appropriate text via the next / previous links on each page. To proceed the poster content click on the icon to the right
