Krapperup's farm museum

These web pages provide translations for the Swedish text shown in the various posters in the museum. At each poster a QR code can be used to access the poster's content in alternative languages. Each poster is numbered and these numbers can be used as an alternative way to find the appropriate text via the next / previous and numeric links on each page. To proceed to the poster content click on the icon to the right.

Krapperups Gutshofmuseum

Diese Webseiten bieten Übersetzungen für den schwedischen Text, der auf den verschiedenen Postern im Museum gezeigt wird. An jedem Plakat kann ein QR-Code verwendet werden, um auf den Inhalt des Posters in alternativen Sprachen zuzugreifen. Jedes Poster ist nummeriert und diese Nummern können als eine alternative Möglichkeit verwendet werden, um den geeigneten Text über die nächsten / vorherigen und numerischen Links auf jeder Seite zu finden. Um zum Inhalt des Posters zu kommen, klicken Sie auf das Symbol rechts.
